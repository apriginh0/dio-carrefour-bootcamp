ESTRUTURA DE REPETIÇÃO
O real poder dos computadores está na sua habilidade para repetir uma operação ou uma série
de operações, várias vezes
Cada repetição é chamada de laço(loop), é um dos conceitos básicos da programação estruturada
Uma estrutura de repetição permite que uma sequência de comandos seja executada
repetidamente, caso determinadas condições sejam satisfeitas
As estruturas de repetição podems er classificadas em:
Repetição com teste no início (while)
Repetição com teste no final (do-while)
Repetição cotada (for)
O comando BREACK é utilizado para terminar de forma abrupta uma repetição
Quando comando CONTINUE é executado, os comandos restantes da repetição são ignorados
e o programa volta a testar novamente ou não

ARRAYS
É um objeto utilizado para armazenar sequencialmente dados do mesmo tipo
